メニュー - システムメニュー

Emulate FDD1 FDD2 HardDisk Screen Device Other

Windowsのシステムメニューの開き方に関してはこまった時に - ウィンドウズを参照してください。

メニュー機能
Tool Windowツールウィンドウを開きます。
Key Displayキーディスプレイウィンドウを開きます。
Soft Keyboardソフトウェアキーボードウィンドウを開きます。
Centeringエミュレーションウィンドウをデスクトップの中央に表示します。
Window Snapエミュレーションウィンドウの移動時にデスクトップの境界近くで引っ張られます。
Backgroundエミュレーションウィンドウがバックグラウンドになっても、エミュレートを続行させます。
Background Soundエミュレーションウィンドウがバックグラウンドになっても、サウンドを再生します。
Restore window borderウィンドウ境界をデフォルトの表示に戻します。
320x200(x0.5)エミュレーション画面サイズをx0.5にします。
480x300(x0.75)エミュレーション画面サイズをx0.75にします。
640x400(x1)エミュレーション画面サイズをx1（等倍）にします。
800x500(x1.25)エミュレーション画面サイズをx1.25にします。
960x600(x1.5)エミュレーション画面サイズをx1.5にします。
1280x600(x2)エミュレーション画面サイズをx2にします。
