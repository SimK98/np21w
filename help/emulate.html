メニュー - Emulate

Emulate FDD1 FDD2 HardDisk Screen Device Other

メニュー機能
Resetハードウェアリセットします。
実機でリセットボタンを押した時の状態になります。
Pauseエミュレーションを一時停止します。
再度選択したりリセットしたりすると解除されます。
Configureエミュレーション設定ダイアログを開きます。
Change CPU Speed仮想マシンのCPUクロック倍率を動的に変更します。本変更は一時的なもので、Resetで復元されます。
Change Emulation Speed仮想マシンの実行速度を変更します。例えばx2にするとエミュレータ内の時間が2倍速で進みます（ホスト性能が十分な場合）。
NewDiskFloppy disk image未フォーマット状態のフロッピーディスクイメージを作成します。
Hard disk image未フォーマット状態のハードディスクイメージを作成します。
Fontエミュレーションで表示するフォントファイルを指定します。
テキストRAM上のデータは即反映されますが、グラフィックRAM上のデータはソフトウェアに依存されてしまいます。
Load VM config仮想マシン設定ファイルを読み込みます。
Save VM config仮想マシン設定ファイルを保存します。
ExitねこープロジェクトIIを終了します。
