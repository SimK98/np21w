IDE HardDisk

メニュー機能
IDE #0OpenIDE #0（プライマリマスタ）に接続するハードディスクまたはCD-ROMイメージを選択します。
RemoveIDE #0を未接続状態（CD-ROMドライブの場合はディスク取り出し状態）にします。
Host Drive x:IDE #0に実CD/DVDドライブx:を割り当てます。
IDE #1OpenIDE #1（プライマリスレーブ）に接続するハードディスクまたはCD-ROMイメージを選択します。
RemoveIDE #1を未接続状態（CD-ROMドライブの場合はディスク取り出し状態）にします。
Host Drive x:IDE #1に実CD/DVDドライブx:を割り当てます。
IDE #2OpenIDE #2（セカンダリマスタ）に接続するハードディスクまたはCD-ROMイメージを選択します。
RemoveIDE #2を未接続状態（CD-ROMドライブの場合はディスク取り出し状態）にします。
Host Drive x:IDE #2に実CD/DVDドライブx:を割り当てます。
IDE #3OpenIDE #3（セカンダリスレーブ）に接続するハードディスクまたはCD-ROMイメージを選択します。
RemoveIDE #3を未接続状態（CD-ROMドライブの場合はディスク取り出し状態）にします。
Host Drive x:IDE #3に実CD/DVDドライブx:を割り当てます。
IDE optionIDE設定ダイアログを開きます。

ねこープロジェクト21/WのIDE HDDでは T98-Next形式(.NHD)、T98形式(.THD)、Anex86形式(.HDI)、Virtual PC形式(.VHD)、SL9821形式(.SLH)等が利用できます。

ねこープロジェクト21/WのIDE CD-ROMでは ISO-9660形式(.ISO)、Cue Sheet形式(.CUE)、CloneCD形式(.CCD)、CD Manipulator形式(.CDM)、Media Descriptor形式(.MDS)、Nero Image形式(.NRG)および実ドライブが利用できます。 ただし、プロテクト付きのCDは扱えません。

HDDイメージの変更はリセット後に有効になります。 リセット時に変更されるイメージファイル名は"旧ファイル名 -> 新ファイル名"の形式でメニューに表示されます。

CDイメージの変更は多少時間がかかります（6秒程度）。

実CD/DVDドライブをマウントしている場合、実CD/DVDのディスク交換を自動的に認識しますのでディスク交換時に再度マウントし直す必要はありません。

SCSI HardDisk

メニュー機能
SCSI #0OpenSCSI #0に接続するハードディスクイメージを選択します。
RemoveSCSI #0を未接続状態にします。
SCSI #1OpenSCSI #1に接続するハードディスクイメージを選択します。
RemoveSCSI #1を未接続状態にします。
SCSI #2OpenSCSI #2に接続するハードディスクイメージを選択します。
RemoveSCSI #2を未接続状態にします。
SCSI #3OpenSCSI #3に接続するハードディスクイメージを選択します。
RemoveSCSI #3を未接続状態にします。

ねこープロジェクト21/WのSCSI HDDでは Virtual98形式(.HDD)、RaSCSI形式(.HDN)が利用できます。 T98-Next形式(.NHD)、Anex86形式(.HDI)、Virtual PC形式(.VHD)も使用できますが、CHSがSCSI互換である必要があります（特にHが15までな点に注意）。

HDDイメージの変更はリセット後に有効になります。 リセット時に変更されるイメージファイル名は"旧ファイル名 -> 新ファイル名"の形式でメニューに表示されます。
