Sound option - JoyPad

Use JoyPad-1

チェックするとジョイパッドを使用します。

Trigger-1
Trigger-2

ジョイパッドのボタンアサインを指定します。

Trigger-1,2はエミュレート中の音源に接続中の、
A/B/C/DはWindowsマシンに接続されているパッドのボタンです。

SB16, 118音源のゲームポートの場合は、この設定によらず番号順に4つのボタンが割り当てられます。

Rapid

チェックしておくと連射状態になります。

POV -> X-Y Axis

ここにチェックを入れるとPOVをX, Y軸操作へ変換します。

JoyPad ID

使用するJoyPadを指定します。

Use Game Port

ゲームポートを有効化します。

Use Analog Input

ゲームポートでアナログX, Y軸入力を可能にします。
