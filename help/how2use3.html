ハードディスクイメージの作成

ねこープロジェクト21/Wでは、フロッピーディスク同様にハードディスクも仮想ディスクファイルよりデータを読み書きを行います。

  1. MS-DOSをディスクイメージにします。
  2. NewDiskを選択し、ファイルの種類(T)から T98-Next harddisk image file (*.NHD)を選択し保存します。
  3. ハードディスクサイズの選択ダイアログが表示されたら 好きな容量をMB単位で設定してください。詳細
  4. MS-DOSのディスクイメージを[FDD1 - Open...]で、
    作成したハードディスクイメージを [HardDisk - IDE #0 - Open] で、それぞれ選択します。
  5. [Emulate - Reset]を選択して、リセットをかけます。
  6. MS-DOSが起動後に表示されるメニューより、MS-DOSのマニュアルに従いハードディスクをフォーマットします。
