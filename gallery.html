価値がありそうなものからないものまで色々

信長の野望　天翔記（DOS版）

GA-98NB系が使えます。パワーアップキットはWAB系も使えます。他、三國志Ⅴも対応しています。

Super MPU(S-MPU)で2ポート32ch MIDI

レコンポーザでも使えます。

Windows2000でSound Blaster 16 for PC-9800

Windows2000でSound Blaster 16 for PC-9800が使えるパッチです。実機でも使えます。INTとCバススロットが余っていてOPL3の音が聞きたい人は試してみても良いかも。

DOS窓でN88-BASIC

DOS窓がちゃんと使えます。直接動かした方が手っ取り早いですけど･･･

FreeBSD4.11 + X Window System (Enlightenment)

とりあえず派手だけど遅い･･･

GA-98NBIVで強引に1600x1024表示

GA-98NBに搭載されているビッグスクリーン機能の表示範囲をいじくって1600x1024をスクロール無しで表示させました。 （ヘルプの非公開機能を参照のこと）

無駄な大容量ディスク

2TBまでアクセスできることを確認しました。が、98では使い切れないでしょう。

Intel Pentium II

MMX搭載してます。しかし、あまり価値はないと思われます。

Quick C

DOSプロンプトでもそれなりに使えます

ReactOS

すごく遅いですが起動できました

Microsoft .NET Framework 2.0 + VB2005

入れるのは大変ですがVisual Basic 2005が動きます。Visual C++ 2005があれば猫で猫をコンパイルできます（無駄）

PC-PR201系・ESC/P系印刷エミュレーション

普通のプリンタでもエミュレーション機能でPC-PR201系やESC/P系のデータ印刷が可能です。PDFプリンタへの印刷もできます。

ドットが好きな人のためのドット印刷板（容量が大きいです）

Neko Project 21/W専用 高速SCSIドライバ

標準IDEよりも高速に動きます。 Win95以降で使用できます。

HOSTDRV for NT

MS-DOS版のHOSTDRV（ホストのディレクトリを仮想ドライブとしてゲストから参照する機能）のWindows NT版です。 ロングファイルネームも使用できます。

ゲームポート

Sound Blaster 16等に搭載のゲームポートを使用できます。 アナログ入力も可能です。 ただしHAXM版では挙動が不安定です。
