Neko Project 21/W（np21w, ねこープロジェクト21/W）はPC-98エミュレータ Neko Project II/21（ねこープロジェクトII/21）をベースに、ニッチなハードウェアのエミュレーションやPC-98後期モデル（1993～2000頃）の再現に耐えうる改造を施したものです。

また、ニッチなハードウェアのエミュレーションに限定してNeko Project II相当まで簡略化したNeko Project II/W（np2w, ねこープロジェクトII/W）も配布しています。CPUが80286（プロテクトモードなし）相当ですが比較的軽量に動きます。

本家猫: http://www.yui.ne.jp/np2/

注意

本エミュレータの機能追加の多くは98らしさが失われていった後期モデル相当ですので、開発者の腕試し的な要素が強く、一般ユーザーにはあまり役に立たないニッチなものです。

本家のNeko Project II/21で動く物は基本的にそのまま動きますが、特別良くなるわけでもありません。 PC-98前期～中期（1993頃迄）のエミュレーションは例えばNeko Project II fmgen版などをおすすめします。

更に前の更新履歴

本家からの変更点

本家から変更されている点は以下の通りです。

CPU／メモリ周辺

  • CPUクロック倍率制限の緩和
  • i486DX以降のCPUを追加（実機であり得ない物も含む）
  • CPU拡張命令の実装（FPU, MMX, SSE, 3DNow! など）
  • 大容量メモリに対応
  • 高速メモリチェック
  • CPUの非同期的動作＋動的なCPUクロック変更

ハードウェア（音源・MIDI）

  • fmgenの部分的サポート
  • MAME OPL音源（118音源とSB16で使用）
  • 118音源の修正
  • Mate-X PCMの追加
  • Sound Blaster 16（PCM可）の追加
  • Super MPU（32ポートMIDI）の追加
  • Q-Vision Wave Starの追加

ハードウェア（グラフィック周辺・ウィンドウアクセラレータ）

  • PEGCプレーンモードの一部
  • PC-9821内蔵 CL-GD54xx ウィンドウアクセラレータ
  • PC-9801-96拡張ボード CL-GD54xx ウィンドウアクセラレータ
  • MELCO WAB-S，WSN-A2F, WSN-A4F ウィンドウアクセラレータ
  • I-O DATA GA-98NBI/C, II, IV ウィンドウアクセラレータ

ハードウェア（その他）

  • 大容量HDDのサポート（4GB未満推奨、最大2TB）
  • IDE 4ポート（HDD または 光学ドライブを任意に選択可）
  • CD-DAの再生
  • 実CD/DVDイメージのマウント（EDC/ECCエミュレーション付き）
  • PCIバス
  • WACOMペンタブレットエミュレーション
  • 名前付きパイプでのRS-232Cエミュレーション

UI周辺／その他

  • 追加のディスクイメージサポート（NVL.DLLで更に追加可能）
  • メニューにマウント中のHDDイメージを表示
  • マウスキャプチャしないでマウス操作（シームレス風味）
  • マウス挙動の修正、マウス速度の指定
  • マスタボリュームの指定
  • Direct3D描画
  • 画面の整数倍拡大、整数倍風拡大（疑似）
  • メニューにマウント中のHDDイメージを表示

バグ修正(本家ver.0.86 基準)

  • CD-ROM認識とCD-DA再生正常化
  • IDE I/Oポートの割り込みタイミング修正
  • HOSTDRVのパーミッションがおかしい不具合を修正
  • EGC周辺の修正
  • RS-232C周辺の大幅修正
