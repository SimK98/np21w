リンク集

PC-98エミュレータの関連リンク集です。

ねこープロジェクト関連

Neko Project II (Windows/MacOS9)
http://www.yui.ne.jp/np2/
本家本元です。開発情報（Redmine）はhttp://amethyst.yui.ne.jp/redmine/へ
Xnp2 (X Window System)
http://www.nonakap.org/np2/
X Window System用のNeko Project IIです。ビルドが必要ですがLinux全般で使えます。
Neko Project II fmgen版 (Windows)
http://nenecchi.kirara.st/
本家猫のFM音源をfmgenに置き換えたりメモリエディタがあったりと色々な改良が行われています。ねこープロジェクト系でDOSベースのアプリケーション・ゲームを動かすなら現時点で最善だと思われます。
Neko Project IIx (MacOSX)
http://retropc.net/tk800/np2x/
MacOS X用のNeko Project IIです。私はMacを持ってないので使ったことがありませんが･･･(^^; (サイト消滅のためInternet Archiveへのリンク)
Neko Project II (NP2) 改変 (SDL2(Windows, UNIX), libretro(Windows, linux, Android等), X11(Linux)など)
http://domisan.sakura.ne.jp/article/np2kai/np2kai.html
本家np2に色々な所に散らばっている亜種のコードを取り込んだものです。np21wのほとんどの機能も取り込まれており、Win以外のOSもサポートされています。なお、別のKai版なるnp2が昔からありますがそれとは別物です。

他のPC-98エミュレータ関連

T98-Next (Windows)
http://akiyuki.boy.jp/old/t98next/
高性能な（FPUとかも付いてます）PC-9821エミュレータです。Newsのページに何かが･･･。
Anex86 (Windows)
（リンクフリーではないためURLは掲載しません）
正確にはエプソン互換機エミュレータです。軽量の割に結構色々動かせる上にプラグインで色々遊べます。現在は公式サイトが消滅していますが、別のサイトで再配布されているらしいです。
QEMU/9821 (Windows)
http://takeda-toshiya.my.coocan.jp/
その名の通りPC-9821エミュレータです。ものすごく高速に動作する代わりに互換性は低め。実機BIOSは必須で、原則としてPC-9801BX4（またはそれに近い機種）のBIOSが必須です。
SL9821 (Windows)
http://www.satotomi.com/sl9821/
初代PC-9821エミュレータです。初期のPC-9821に搭載されていたPEGCプレーンモードが再現されています。使用できる実機BIOSはQEMU/9821よりもさらに限定的です。とりあえず後期PC-9801のBIOSも使えるようですが、HDDからブートできない等の問題が起こります。

ツール関連

DiskExplorer (Windows)
http://hp.vector.co.jp/authors/VA013937/editdisk/
各種PC-98エミュレータのディスクイメージを編集できます。妙な形式のイメージファイルでも手動サーチで読める場合があります。FAT32やNTFSは使えません。
ND (Windows), NDL (Linux)
http://euee.web.fc2.com/tool/nd.html, http://euee.web.fc2.com/tool/ndl.html
各種エミュレータのディスクイメージを編集できます。一応LongFileNameにも対応していて、VMwareやVirtualPC形式も読めます。HDDについてはFAT16/FAT32は編集できそうですがNTFSは使えません。 詳細はhttp://euee.web.fc2.com/ へ
Virtual Floppy Image Converter (Windows)
http://www.vector.co.jp/soft/win95/util/se151106.html
各種PC-98エミュレータのFDイメージを変換出来ます。
NHC (Windows, Linux)
http://euee.web.fc2.com/tool/nhc.html
各種エミュレータのディスクイメージを変換できます。VMwareやVirtualPC形式もOKです。 詳細はhttp://euee.web.fc2.com/ へ
